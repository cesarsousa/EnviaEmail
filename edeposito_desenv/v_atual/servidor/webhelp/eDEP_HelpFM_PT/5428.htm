Glossário

ASCII

Código Padrão Americano para Intercâmbio de Informações.

Pode conter letras, números, espaços e pontuação, mas não formatação. Também chamado de arquivo de texto.

CD-ROM

Um CD-ROM (Compact Disc Read-Only Memory) é um CD que armazena permanentemente dados gravados. Um CD-RW, em contraste, é regravável (RW), o que significa que permite a supressão de dados do CD.

Dígito de verificação

Um dígito de verificação contém um algoritmo que verifica números introduzidos, ajudando a reduzir erros de digitação.

CORBA

(Common Object Request Broker Architecture): arquitetura que define protocolos multi-plataforma e serviços, facilita o desenvolvimento de aplicações distribuídas e aplicações em ambientes heterogêneos.

CSV

(Comma Separated Values) .

Um formato de arquivo normalmente utilizados para conjuntos de dados onde os dados são dispostos em colunas e linhas. Os campos de dados individuais são separados uns dos outros por delimitadores, tais como vírgulas.

Padrão

Seleção automática feita pelo sistema quando o usuário não seleciona uma alternativa específica.

Delimitador

Um caracter especial que define, ou separa, individualmente ítens de um conjunto de dados. Vírgulas e ponto e vírgula são exemplos de delimitadores comumente utilizados.

Ancorar

Para mover uma barra de ferramentas ou janela para a borda de uma janela do aplicativo complementando-se a ela.

DVD-ROM

Um DVD (Digital Video Disc) é uma unidade de armazenamento óptico de dados como um CD, mas com capacidade significativamente maior (cerca de 4 GB). Por conseguinte, é principalmente utilizada para o armazenamento de filmes de vídeo. DVD-ROM (Read-Only Memory) são geralmente usados para guardar dados. Um gravador de DVD adequado é necessário para gravar dados para um DVD.

Campo

Um espaço em um formulário na tela onde o usuário pode inserir algum item específico de informação, como por exemplo um nome ou uma data. Campos podem ter restrições quanto ao tamanho e tipo de dados que podem ser inseridos, por exemplo texto ou apenas números em um determinado formato.

Firebird servidor de banco de dados SQL

Firebird é uma versão simplificada de um sistema de gerenciamento de dados relacional produzido pela Borland.

GUI

(Graphical User Interface)  Interface Gráfica do Usuário: componente de software do usuário, que permite a interação com os elementos gráficos no programa através de um mouse e um teclado.

HTTP

HyperText Transfer Protocol: protocolo de transferência de informações na WWW, geralmente em HTML.

HTTPS

HyperText Transfer Protocol Secure: indica que o HTTP é usado com uma porta diferente (443) e uma camada de criptografia/autenticação adicional entre HTTP e TCP.

Ícone

Uma pequena imagem exibida na tela que permite ao usuário controlar determinadas ações do computador  sem ter que lembrar comandos ou digitá-los no teclado.

IIOP

Internet Inter ORB Protocol: utilizado no CORBA, usado para executar RPCs (Chamadas de procedimento remoto).

LAN

Rede de Área Local: normalmente dentro de uma empresa ou pequeno escritório.

Não-repúdio (Non-repudiation)

Para proteger e garantir a confiança nas assinaturas digitais, as partes podem empregar o chamado "não-repúdio", que não apenas valida o remetente, mas também a data e hora da transação, de modo que não se possa posteriormente reivindicar que a transação não foi autorizada ou não era válida.

ORB

Object Request Broker: agente que permite a comunicação entre objetos dentro de um sistema distribuído, como por exemplo a Internet.

Patch

Uma nova característica ou função adicionada a um programa, normalmente utilizados como uma medida provisória antes do lançamento de uma versão completa do software.

PDF

Portable Document Format.

Um formato de arquivo usado para salvar documentos. Ele pode ser lido utilizando algum software leitor de PDFs.

PIN

Número de Identificação Pessoal: uma senha numérica compartilhada entre um usuário e um sistema, usado para autenticar o usuário no sistema.

PMS

Sistema de Gestão de Patentes.

Porta

Uma porta é parte de um endereço de rede para um aplicativo servidor executado em um computador dentro da rede. Portas típicas são 80 para Servifores Web HTTP e 110 para servidores de correio POP3.

Servidor Proxy

Um componente de firewall que gerencia o tráfego de Internet de e para uma rede.

SOAP

Simple Object Access Protocol: protocolo que usa XML para visualização de dados e principalmente HTTP para a transmissão.

 

SQL

Linguagem de Consulta Simples: usada para consultar e modificar bases de dados.

SSL

Secure Socket Layer.

Um protocolo desenvolvido pela Netscape Communications Corporation para garantir a segurança e privacidade nas comunicações na Internet. Suporta autenticação de cliente, servidor, ou ambos, bem como criptografia durante uma sessão de comunicação.

TCP

Transmission Control Protocol - Protocolo de Controle de Transmissão

Protocolo utilizado para gerenciar a troca de dados entre computadores através de conexões da rede.

TIFF

Tagged Image File Format.

Um formato de arquivo padrão comumente utilizado para digitalização e armazenamento em escala de cinza imagens. TIF podem ser utilizados para as assinaturas digitalizadas.

URL

Uniform Resource Locator.

Um endereço de algum recurso/site na internet. Usado pelos navegadores para localizar recursos/sites da Internet.

WAN

Wide Area Network: Rede normalmente existente em vários escritórios ou filiais de uma grande empresa ou organização, incluindo partes de conexão via Internet entre eles.

 

Fluxo de Trabalho

Corresponde ao monitoramento e gerenciamento de todas as atividades do início ao fim.

XML

eXtensible Markup Language.

XML é um padrão de processamento de documentos oficialmente recomendado pelo Consórcio W3C, amplamente aceito pela comunidade que utiliza a Internet. Ele fornece uma maneira de definir e gerenciar informações, podendo ser usado para verificar a precisão e a qualidade dos documentos.
