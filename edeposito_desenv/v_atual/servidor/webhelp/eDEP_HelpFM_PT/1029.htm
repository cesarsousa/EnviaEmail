Exportando dados como XML para pastas

Petições individuais do e-Patentes/Depósito podem ser exportados como arquivos XML para uma pasta externa. Esta operação pode ser realizada a partir do e-Patentes/Depósito Editor ou da Janela do formuláriontes.

  • Você pode selecionar várias petições, uma após a outra, clicando-as com a tecla CTRL pressionada.
  • Exportar a partir da Janela do Formulário só é possível enquanto a aplicação ainda estiver no status Rascunho.
  • No e-Patentes/Depósito Editor, selecione as petições.
  • Selecione Exportar > Petição para uma pasta a partir do menu Arquivo no e-Patentes/Depósito Editor.

    -OU-

    Na janela do Formulário, selecione Arquivo > Petição para uma pasta.

O sistema selecionará inicialmente a localização escolhida nas Preferências do Usuários.

  • Clique OK para iniciar a exportação.

  • Clique OK novamente no aviso de Exportação completa.

Uma nova pasta é criada automaticamente com o nome equivalente ao Código de Referência da petição.

Se o diretório de destino já contém uma pasta com o mesmo nome, ela será substituída sem aviso prévio.

A nova pasta contém os arquivos XML e PDF criados pelo e-Patentes/Depósito.
