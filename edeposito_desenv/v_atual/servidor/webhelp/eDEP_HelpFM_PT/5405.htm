Alterando o status de uma petição no formulário

Uma petição permanecerá no status Rascunho enquanto não for, pelo usuário, movida para o próximo status, e.g. movida para o status de Pronto para Assinar .

  • Abra uma petição no status Rascunho.
  • Na barra de ferramentas, clique no botão Próximo.

-OU-

  • Selecione Fechar no menu Arquivo e a opção Mover para "Pronto para assinar" e sair.

Quando uma petição deixa o status de Rascunho, o próximo passo no processo de fluxo de trabalho é iniciado automaticamente na próxima vez que a petição é aberta.
Por exemplo, o processo de assinatura inicia automaticamente ao se abrir uma petição que esteja no status Pronto para Assinar.  Ao se abrir uma petição no status Pronto para Enviar, o processo de envio se inicia.
Use o botão de fluxo de trabalho Anterior no  e-Patentes/Depósito Editor para mover a petição de volta ao status de Rascunho, a fim de fazer mudanças no modo de formulário, caso desejado.
