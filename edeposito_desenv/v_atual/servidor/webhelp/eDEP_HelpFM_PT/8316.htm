Desbloqueando usuários

O e-Patentes/Depósito desativa automaticamente um usuário após três tentativas sucessivas de logon com a senha errada. O Administrador ou outro usuário que é membro do grupo de Administradores pode desbloquear este usuário para deixá-lo fazer logon no e-Patentes/Depósito novamente.

Se um usuário é bloqueado após três tentativas frustradas de digitar a senha correta, a seguinte caixa de mensagem aparece na próxima vez que ele tentar fazer logon:

ATENÇÃO: Certifique-se de criar um segundo usuário com direitos totais de administrador, ou seja, com autorização para fazer a administração do usuário. O nome do administrador do usuário é bloqueado após três tentativas sucessivas de logon com a senha errada. Se isso acontecer, não há nenhuma maneira para redefinir a senha, a menos que haja outro usuário com direitos de administrador configuradas em seu sistema.

Desbloqueando a conta de um usuário

A caixa de seleção A conta do usuários não está habilitada fica marcada após sucessivas tentativas de login.

  • Desmarque a caixa de seleção.
  • Clique em Salvar tudo.

A conta de administrador também pode ser bloqueada após três tentativas fracassadas de logon. Para desbloquear a conta, outro usuário com privilégios de administração do usuário deve fazer logon no Editor de e-Patentes/Depósito. A seguinte mensagem aparece ao abrir a Gestão de Usuários:

  • Clique em OK.

    A janela Gestão de usuários abre. A  caixa de seleção A conta do usuários não está habilitada automaticamente é desmarcada.

  • Clique em Salvar tudo para completar a operação e reativar a conta do administrador.

Bloquenado a conta de um usuário

O administrador pode, se necessário, também desativar um usuário específico, assumindo-se que exclusão permanente do usuário não é necessária, somente o seu bloqueio.

  • Clique na caixa A conta do usuários não está habilitada.
  • Clique em Salvar tudo.
