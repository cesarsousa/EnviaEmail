Criando um novo modelo

Você pode criar um modelo a partir do rascunho de uma petição, a partir de um formulário em branco ou usando um modelo já existente.

Criando um modelo a partir de um formulário em branco

  • Selecione Novo Modelo no menu Arquivo File do e-Patentes/Depósito Editor.

    -OU-

    Presione SHIFT+CTRL+N.

  • Na janela Criar Novo Modelo, entre com o Nome do Modelo.
  • Selecione o Procedimento  e o Idioma do Formulário.
  • Opcionalmente, entre com uma descrição para auxiliar na identificação do modelo.

    Será mostrado na coluna Descrição no e-Patentes/Depósito Editor.

  • Clique em Criar.

Criando um modelo a partir de uma petição existente (rascunho).

A petição está com o status de Rascunho.

  • Abra a petição desejada com um duplo clique.
  • Selecione Salvar como > Salavar como Modelo no menu Arquivo da barra de ferramentas.
  • Entre com o  Nome do Modelo na janela Salavar como modelo.
  • Selecione a Pasta caso se queira salva em uma sub-pasta da pasta Modelo.

    Clique duplo na pasta Modelo para abrir todas as sub-pastas.

  • Entre com a descrição.
  • Clique em Salvar.

Convertendo uma petição (rascunho) em um modelo

A petição está com o status de Rascunho.

  • Clique com o botão direito na petição que se deseja converter em modelo no e-Patentes/Depósito Editor.
  • Selecione Converter em modelo no menu.
  • Selecione uma pasta na janela Converter em modelo e entre a descrição.

    Não é possível alterar o Nome do Modelo aqui.

  • Clicar em Salvar.

O modelo aparecerá na pasta Modelos do e-Patentes/Depósito Editor.

Criando um modelo a partir de uma petição já processada (Enviado, Pronto para Enviar, Pronto para Assinar)

Somente as petições ainda com o status de Rascunho podem ser convertidas diretamente em modelos. Há, no entanto, também uma forma de criar um modelo a partir de uma petição, por exemplo, que já tenha sido enviada.

  • No e-Patentes/Depósito Editor, selecione a petição que se deseja utilizar como modelo.

    Seu status pode ser Enviado, Pronto para Enviar ou Pronto para Assinar.

  • Clique com o botão direito e selecione Copiar no menu.
  • Digite um novo Nome do Modelo.
  • Clique em Copiar.

    A copia aparecerá com o status Rascunho no e-Patentes/Depósito Editor.

  • Clique com o botão direito e selecione Converter em modelo no menu.
  • Para continuar, siga o procedimento descrito acima.

Copiando um modelo

  • Clique na pasta Modelos.
  • Selecione o modelo que se quer copiar.
  • Clique com o botão direito e selecione Criar uma cópia  do modelo no menu.
  • Entre com o nome para o novo modelo na janela Copiar modelo.
  • Clique em Copiar.

No e-Patentes/Depósito Editor, o novo modelo aparece na mesma pasta que o modelo copiado.
