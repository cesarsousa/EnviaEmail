Configurações de rede

A configuração de rede depende de como a rede de computadores está configurada em sua empresa ou casa. Normalmente é suficiente selecionar a caixa de seleção Utilizar os parâmetros padrões de rede para estabelecer a conexão do e-Patentes/Depósito com a Internet usando as configurações do Painel de Controle do Windows.

Alternativamente, você pode digitar um endereço IP ou o nome do servidor no campo Servidor proxy e o número da porta do servidor proxy que você deseja usar para estabelecer conexões de internet no campo após os dois pontos. Você pode selecionar a versão de criptografia para ser usado em sua rede no campo Versão SSL - este deve corresponder as configurações do servidor OLF.

Você pode ser solicitado a inserir um nome de usuário e senha para registro no servidor proxy.

Por favor, entre em contato com o administrador do seu sistema se não tiver certeza sobre a necessidade da alteração dos parâmetros de rede.
