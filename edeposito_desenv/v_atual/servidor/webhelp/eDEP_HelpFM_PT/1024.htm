e-Patentes/Depósito inicia com a janela do e-Patentes/Depósito Editor. A pasta Formulários é sempre mostrada quando o aplicativo é iniciado, permitindo que você selecione o formulário desejado e inicie um novo depósito ou petição.

  • Clique Todas as petições, para ver a lista dos formulários  e navegar pelos elementos descritos abaixo.

Elemento

Comentário

Barra de Menu

Contém todos os comandos que você precisa para criar, editar e salvar petições, opções para configurar as propriedades do programa de e-Patentes/Depósito bem como outras ferramentas.

Barra de Ferramentas

Fornece atalhos para as tarefas mais utilizadas e ferramentas.

Barra Indicadora de Localização

Contém informações específicas sobre a petição selecionada: pasta, estado, usuário, código de referência.

Barra de Seleção de Status

Exibe as petições na pasta selecionada filtrada por seu status atual: Rascunho, Pronto para Assinar, Pronto para Enviar ou Enviado.

Busca

Procura na lista de petições (ou modelos) um termo de pesquisa específico. Para buscar um termo, selecione, na lista, a coluna que você deseja buscar e digite o termo no campo a direita.
A primeira petição (ou modelo) da lista que coincide com os termos da pesquisa é destacado em cinza, após clicar no botão Buscar  . Clique em Buscar novamente para pular para a próxima petição que coincide com os critérios de busca.

Pastas e Modelos

Contem as pasta do sistema: Todas as petições, Pasta Padrão, Formulários, Modelos e Lixeira. Você pode criar pastas e subpastas adicionais, conforme necessário, para as suas petições e modelos.

Painel de Informações

Exibe informações gerais sobre a petição (ou modelo), atualmente em destaque na lista.

Clique no butão Visualizar para ter um preview da petição. A petição, então, aparece no visualizador de PDF com uma lista de todos os arquivos anexados, que também podem ser exibidos no modo PDF.

Lista de Petições

Mostra o conteúdo da pasta que você selecionou na esquerda, isto é, todas as petições, modelos ou formulários que estão nesta pasta.

Barra de Status

Mostra informações sobre a última ação executada no programa.

Customisando e-Patentes/Depósito Editor para mostrar as opções que se deseja ver

Você pode personalizar o e-Patentes/Depósito Editor para exibir apenas os elementos que você deseja ver.

  • Clique nas opções marcadas no menu Ver  uma a uma, e.g. Barra de Ferramentas e Indicador de localização.

    As marcações sendo removidas os elementos Barra de Ferramentas e Indicador de localização ficarão ocultos.

  • Clique em uma opção no menu Ver novamente para exibir o elemento mais uma vez no e-Patentes/Depósito Editor.

Saindo do e-Patentes/Depósito Editor

Utilize o menu Arquivo para fechar o e-Patentes/Depósito Editor e sair do e-Patentes/Depósito.

  • Selecione Sair para fechar o programa.
  • Selecione Fechar Sessão se você deseja utilizar o programa com um nome de usuário diferente.
