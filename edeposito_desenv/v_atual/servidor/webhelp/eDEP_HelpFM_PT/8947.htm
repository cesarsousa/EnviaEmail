Atualizando o programa de e-Patentes/Depósito

O INPI está constantemente desenvolvendo e atualizando do sistema e-Patentes/Depósito em resposta às necessidades dos clientes para melhorar o desempenho, qualidade de dados e fluxo de trabalho.

Todas as atualizações do programa estão disponíveis para download automático através do serviço Atualização Automática, e os arquivos de atualização estão disponíveis no site do INPI para download manual.

Atualização Automática

O INPI recomenda que todos os usuários do programa configurem a função de Atualização Automática. Isso garante que você sempre esteja sempre usando a versão mais recente do e-Patentes/Depósito, incorporando as mais recentes mudanças nos formulários.

A Atualização automática também permite que se faça o download e a instalação de atualizações específicas para cada formulário.

Para obter detalhes sobre como usar e configurar esta ferramenta, consulte a seção de Atualização Automática mais adiante neste manual.

Atualização manual

A listagem cronológica de todas as atualizações disponíveis para download  pode ser encontradas no site do ePatentes na internet (http://epatentes.inpi.gov.br/modulo2/edeposito/).
