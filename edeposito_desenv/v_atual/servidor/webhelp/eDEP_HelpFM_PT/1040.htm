A Agenda facilita a armazenagem de nomes e detalhes de contatos (e.g. depositantes, procuradores, inventores).

Primeiro abra o e-Patentes/Depósito Editor para ativar a agenda.

  • Selecione Ferramentas > Agenda a partir do menu do e-Patentes/Depósito Editor.

    -OU-

    Clique no botão Agenda na barra de ferramentas.

Os contatos na agenda são agrupados em pessoas físicas e jurídicas. Observe que as Pessoas Jurídicas são representadas pelo ícone   e as Pessoas Físicas pelo ícone .

Funções e botões da agenda

Opção

Comentário

Adicionar

Clique para criar um novo contato na agenda.

Remover

Clique para apagar um contato na agenda.

Desfazer alterações

Clique para cancelar modificações ou entradas.

Buscar

Selecione o campo que você deseja pesquisar lista Buscar por e digite o termo de pesquisa relevante no campo de busca.  Clique para encontra um contato que satisfaça os critérios de busca.

Entrada de detalhes

Editar os detalhes dos contatos nos campos de dados. Você deve digitar pelo menos um nome e sobrenome de uma pessoa física, ou pelo menos o nome da empresa para uma pessoa jurídica.

Número dos documentos de identificação

Neste campo é possível armazenar os seguintes número de identificação: CPF (número do Cadastro de Pessoas Físicas); CNPJ (número do Cadastro Nacional de Pessoa Jurídica); API (número de registro de Agente da Propriedade Industrial, emitido pelo INPI) e OAB (número de registro de Advogado, emitido pela Ordem dos Advogados do Brasil)

Troca de dados entre a Agenda e as petições/formulários

Ao preencher as petições, você pode copiar as entradas da agenda para as pessoas listadas na guia Nomes. Você também pode copiar as informações sobre uma pessoa que você entrou em uma petição para uma nova entrada na agenda.

Troca de dados entre a Agenda e arquivos externos

Você pode criar contatos na Agenda diretamente no e-Patentes/Depósito ou importá-los a partir de arquivos existentes CSV. Você também pode exportar contatos criados no e-Patentes/Depósito para um arquivo CSV externo.
