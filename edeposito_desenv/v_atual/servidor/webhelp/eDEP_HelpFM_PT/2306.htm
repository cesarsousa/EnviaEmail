Assinando petições

Uma vez que todas as informações obrigatórias foram preenchidas, a petição pode ser assinada. O processo de assinatura pode ser iniciado a partir da Janela do Formulário ou a partir do e-Patentes/Depósito Editor. Uma assinatura digital é necessária no formulário. Você deve ter autorização para assinar os formulários.

Preparando para assinar

Se você quiser assinar uma petição, você deve primeiro movê-la para o status Pronto para Assinar no fluxo de trabalho. Há duas maneiras de fazer isso se a petição ainda ainda estiver no status Rascunho:

  • Selecione Salvar como > Pronto para assinar no menu Arquivo, com a petição ainda aberta. A janela do formulário fecha, movendo a petição para o status Pronta para assinar no e-Patentes/Depósito Editor.

    -OU-

  • Selecione a petição no e-Patentes/Depósito Editor e clique no botão Próximo. (a dica mostra "Pronto para assinar")

Iniciando o processo de assinatura no e-Patentes/Depósito Editor

Se o status da petição estiver em  Pronto para assinar, o processo de assinatura é iniciado automaticamente como o próximo estágio no processo de fluxo de trabalho, logo que você abra a petição ou clique no botão Avançar.
 

  • Abra a petição no e-Patentes/Depósito Editor com um duplo clique.

    -OU-

  • Selecione a petição no e-Patentes/Depósito Editor e clique no botão Avançar. (a dica mostra "Assinar").

Assinando

O Visualizador de Documentos abre com uma pré-visualização da petição logo que se altere o status para Assinar. Se desejar, você pode verificar todos os documentos, mais uma vez, antes de serem assinados.

  • Clique Assinar agora no botão abaixo e a direita do Visualizador de documentos.

    Clique em Cancelar  para interromper o processo de assinatura.

A janela  Assinar petição abre.

  • Selecione um dos nomes listados como requerentes ou procuradores.
  • Selecione o tipo de assinatura desejada.

    O tipo de assinatura aqui é o definido como padrão em Preferências do usuário. Se você quiser alterar permanentemente o padrão, selecione o tipo de assinatura que você quer e ative a caixa de seleção Guardar configuração na parte inferior da janela.

  • Caso tenha sido selecionado Certificado A1, será necessário selecionar o arquivo que contem o certificado por software, através do botão Escolher arquivo.
  • Finalmente, clique Assinar.

Removendo a assinatura

  • Selecione a petição que deseja que a assinatura seja removida.
  • Clique como botão direito e selecione Apagar assinaturas
