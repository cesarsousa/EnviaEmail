Exportando dados como XML para arquivos ZIP

Petições individuais podem ser exportadas como arquivos XML no formato compactado ZIP. Esta operação pode ser realizada a partir do e-Patentes/Depósito Editor ou da Janela do formulário.

  • Você pode selecionar várias petições, uma após a outra, clicando-as com a tecla CTRL pressionada.
  • Exportar a partir da Janela do Formulário só é possível enquanto a aplicação ainda estiver no status Rascunho.
  • No e-Patentes/Depósito Editor, selecione as petições.
  • Selecione Exportar > Petição para um arquivo ZIP a partir do menu Arquivo no e-Patentes/Depósito Editor.

    -OU-

    Na janela do Formulário, selecione Arquivo > Petição para um arquivo ZIP.

    O sistema selecionará inicialmente a localização escolhida nas Preferências do Usuários.

  • Altere o Nome do arquivo se necessário se você não quiser usar o Código de Referência da petição por padrão.
  • Clique Salvar para iniciar a exportação.

    O arquivo será salvo no formato ZIP.
