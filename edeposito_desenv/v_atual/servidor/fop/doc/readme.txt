Readme to deploy the OLF Portal Client Package
========================================

This readme describes how to deploy the OLF Portal Client Package using the supplied scripts.

1. Step: Update the stage descriptor
------------------------------------
Set the following stage descriptor keys:


FOP_HOME=
JAVA_HOME=
SYSDIR=

For further information please also take a look into the stage descriptor (../etc/stage) which is documented.


2. Step: Deploy script
----------------------------

Change to ../bin directory and execute ./deploy.sh  scripts
