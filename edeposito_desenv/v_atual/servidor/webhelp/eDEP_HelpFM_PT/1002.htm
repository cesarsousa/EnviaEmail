Atualização Online

A janela de Preferências do Sistema permite que você faça as configurações gerais que se aplicam a todos os usuários do e-Patentes/Depósito.

As preferências do sistema só podem ser modificadas por usuários pertencentes a um grupo com um perfil adequado de privilégios. Você será notificado se o seu nome de usuário não está autorizado a modificar as preferências do sistema.

Você deve reiniciar o  e-Patentes/Depósito Editor cada vez que você clicar em Salvar para aplicar as configurações feitas.

  • No e-Patentes/Depósito Editor selecione Preferências > Preferências do Sistema a partir do menu Ferramentas.

    -OU-

    Clique no botão Preferências do sistema.

Na janela de Preferências do sistema, as configurações básicas estão listadas no item Geral.

Versão e data de instalação

Ao contactar a equipe de Suporte ao Usuário INPI, é sempre útil ser capaz de especificar a versão ou número de compilação do software instalado como exibida na parte superior da tela. Os módulos do e-Patentes/Depósito Editor e os formulários pode ter números de versão diferentes, dependendo de quando eles foram instalados ou atualizados.

Segurança e Gestão de Usuários

Somente o Administrador pode modificar a opção Habilitar Gestão de Usuários. A configuração padrão é selecionado pela primeira vez quando e-Patentes/Depósito está instalado. Se a Gestão de usuários é ativado, todos os usuários devem digitar seu nome de usuário e senha para se conectar ao e-Patentes/Depósito Editor.

A opção Ativar administrador de diretórios  gerencia se a aba Pastas estará ativa. Essa opção não é habilitada automaticamente.
