Salvando petições

Uma nova petição só é guardada no banco de dados após o rascunho ter sido salvo pela primeira vez. As alterações subseqüentes feitas no processo e no fluxo de trabalho podem ser salvas a qualquer momento. Salvar as alterações garante que os dados exibidos no  e-Patentes/Depósito Editor estarão sempre atualizados.

  • Selecio Salvar no menu Arquivo.

    -OU-

    Clique no botão Salvar na barra de ferramentas.

Pode ser utilizada a opção Salvar como  na barra de ferramentas para criar uma nova petição a partir da petição aberta:

  • Selecione Salvar como rascunho e entre com uma nova Referência de Usuário, selecionando, em seguida, a pasta em que se deseja salvar.

    O original será mantido com a última modificação salva. Todas as modificações subseqüentes serão no novo documento.

  • Selecione Salvar como modelo para salvar um formulário customizado que poderá servir de base para petições posteriores.
