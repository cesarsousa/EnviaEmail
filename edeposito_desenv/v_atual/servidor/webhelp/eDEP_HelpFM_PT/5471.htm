Modificando um modelo

Não é possível alterar o tipo de petição do modelo. Também não se pode mudar o nome de um modelo. As alterações feitas a um modelo não são transferidos para as petições existentes com base neste modelo.

Editando dados em um modelo

  • Clique na pasta Modelos.
  • Selecione o modelo que se quer editar.
  • Clique com o botão direito no modelo e selecione Editar Modelo no menu.

    O modelo será aberto em uma nova janela de petição.

  • Modifique os dados desejados e salve o modelo.

Modifique a descrição de um modelo

  • Clique na pasta Modelos.
  • Selecione o modelo que se quer editar.
  • Clique com o botão direito no modelo e selecione Propriedades no menu.
  • Altere o texto no campo Descrição do Modelo na janela Propriedades do Modelo.
  • Clique Salvar.
