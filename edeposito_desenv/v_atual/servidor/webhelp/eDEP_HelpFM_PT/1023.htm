Pastas

O e-Patentes/Depósito Editor   apresenta uma série de pastas especiais do sistema para ajudar a organizar o seu trabalho. Você não pode renomear ou apagar estas pastas do sistema.

Ícone

Pasta do Sistema

Comentário

Todas as Petições

Contém uma lista de todos as petições (além de objetos da pasta Lixo).

Pasta Padrão

Todos os novos rascunhos são arquivados aqui, a menos que você especifique um local diferente ao salvar.

Formulários

Contém os formulários oficiais previstos para os diversos procedimentos que estão disponíveis para o peticionamento eletrônico. A lista de petições disponíveis depende das configurações escolhidas durante a instalação, e das petições já disponibilizadas pelo INPI.

Modelos

Contém formulários modificadas com informações específicas inseridas pelo usuário.

Lixeira

Contém petições (ou modelos) excluídos. Uma vez que a pasta Lixeira for esvaziada, nenhum dos objetos excluídos no e-Patentes/Depósito poderá ser recuperado.

Você pode criar suas próprias pastas e sub-pastas no e-Patentes/Depósito Editor. Essas pastas podem ser nomeadas por tipo de formulário, nome da empresa, ou qualquer outro critério escolhido. As pastas são ordenadas alfabeticamente abaixo da Pasta Padrão. Você pode renomear e excluir as pastas criadas.

Criando uma pasta

  • No e-Patentes/Depósito Editor, selecione Nova Pasta no menu Arquivo.

    -OU-

  • Clique com o botão direito na área das pastas e selecione Nova Pasta a partir do menu de atalho.
  • Digite o nome para a pasta e pressione ENTER.

Criando uma subpasta

  • Clique com o botão direito na pasta que se deseja adicionar uma sub-pasta.
  • Selecione Nova Subpasta no menu de atalho.
  • Digite o nome para a subpasta e pressione ENTER.

Renomeando uma pasta

  • Clique com o botão direito na pasta que deseja renomear.

    -OU-

    Pressione a tecla F2.

  • Selecione Renomear Diretório no menu de atalho.
  • Digite o novo nome e pressione ENTER.

Excluindo uma pasta

  • Clique com o botão direito na pasta que se deseja apagar.
  • Selecione Apagar Diretório no menu de atalho.

    Esta pasta será apagada imediatamente.

Por questões de segurança, você só pode excluir pastas que não contêm quaisquer petições ou modelos.

Ordenando as pasta

  • No e-Patentes/Depósito Editor, selecione a pasta que deseja ordenar.
  • Selecione o nome da coluna que se deseja ordenarr em uma lista no menu Ver > Ordenar.

-OU-

  • Clique no cabeçalho da coluna na lista de petições.
  • Clique novamente no cabeçalho da coluna para alterar a ordem de ascendente (A-Z ou 0-9) para descendente (Z-A ou 9-0).
